# KAI

KAI is a modular AI system designed to evolve from a stateless chatbot into a persistent, memory-driven assistant.

It combines:
- Memory (long-term context)
- Reasoning (planning and decision-making)
- Tools (APIs and browser automation)

## Goal
To build an assistant that remembers users, uses tools, and improves over time as a continuous system rather than a single-session chatbot.

## Status
Early development. Core architecture and initial system structure are being built.
